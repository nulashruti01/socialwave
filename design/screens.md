# Screen Specs — SocialWave

Each screen below maps the requested UI/UX features to layout blocks and interactions.

## 1 — Splash Screen
- Center animated logo (Lottie) with subtle neon pulse
- Background: `--sw-gradient-midnight` with glass vignette
- Transition: fade -> auth or home

## 2 — Authentication
- Login: email/password, social login (Apple/Google), forgot password link
- Sign up: progressive multi-step with avatar upload
- Microinteractions: input lift, button ripple, success toast

## 3 — Home Feed
- Stories carousel (sticky)
- Infinite post feed with skeleton loaders
- Actions: like, comment, share, save
- Double-tap like animation, long-press quick actions
- Reels preview: small vertical preview cards inline

## 4 — Reels
- Fullscreen vertical video; dark cinematic overlays
- Floating FABs for actions, music pill overlay
- Gestures: swipe for next, double-tap like

## 5 — Explore
- Masonry grid (2 columns), dynamic cell heights
- Search bar with AI suggestion chips
- Filters panel and category chips

## 6 — Profile
- Editable header, story highlights, segmented tabs
- Creator dashboard card with metrics and actions

## 7 — Messaging
- Thread list; chat screen with bubble UI
- Voice notes and image sharing, typing indicators
- Video call button and live call state

## 8 — Notifications
- Activity grouped by type and time
- Follow requests with action buttons

## 9 — Create Post
- Stepper modal: upload -> edit -> caption -> tags -> preview
- Multi-image carousel with drag reorder
- Editing tools and AI caption/hastags

## 10 — Settings
- Theme toggle, privacy flags, notification preferences
- Accessibility options (reduce motion, font size)

---

Each screen should be designed as an iPhone 14 Pro and Pixel 6 artboard. Exportable layers and redline spacing notes should be included in the final design pack.
